# FashionablyLate お問い合わせ管理アプリ

## アプリケーション概要

FashionablyLateは、お問い合わせフォームから送信された内容をデータベースに保存し、管理者がログイン後にお問い合わせ内容の一覧表示、検索、詳細確認、削除、CSVエクスポートを行えるお問い合わせ管理アプリケーションです。

## 作成目的

ユーザーからのお問い合わせ内容を管理者が効率的に確認・管理できるようにするために作成しました。

主な目的は以下の通りです。

- ユーザーがお問い合わせ内容を入力・確認・送信できる
- 管理者が会員登録・ログインできる
- 管理者が送信されたお問い合わせ内容を一覧で確認できる
- 管理者が条件検索、詳細確認、削除、CSV出力を行える

## アプリケーションURL

| 画面 | URL |

- ログアウト

- お問い合わせ詳細モーダル表示
- お問い合わせ削除
- CSVエクスポート
- ページネーション
